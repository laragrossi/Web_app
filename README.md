# web_app
Integrante: Lara Lima Grossi
Turma: 3°DS (Desenvolvimento de Sistemas)
Instituição: EEEMI Prof. Joaquim de Moura Candelária
O projeto consiste em um sistema de login que incorpora recursos adicionais de segurança. Entre suas principais funcionalidades estão:
- Autenticação de usuários com verificação de senha;
- Bloqueio automático após três tentativas incorretas;
- Contador de acessos para cada usuário;
- Identificação de primeiro acesso;
- Possibilidade de cadastrar novos usuários diretamente pelo aplicativo.

Tecnologias utilizadas:
Front-end: HTML, CSS e JavaScript
Back-end: PHP
